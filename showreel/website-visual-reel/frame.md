# Hsin-Hsin Yuan Website Visual Reel

## Concept

A 30-second website-first showreel for a documentary director and cross-cultural storyteller. The reel should feel international, cinematic, editorial, and useful for work inquiries without becoming a hard sales ad.

## Format

- Landscape: 1920 x 1080
- Duration: 30 seconds
- No narration in this first version
- Built from current website assets, still images, platform names, impact metrics, and contact copy

## Visual Identity

- Background: `#0B0B0C`
- Foreground: `#F7F2E8`
- Muted text: `#B8B0A3`
- Accent acid: `#D8FF3E`
- Accent heat: `#FF4D1F`
- Accent blue: `#7CC7FF`
- Typography: `League Gothic` for large display, `Montserrat` for editorial body, `JetBrains Mono` for metadata and proof points

## Direction

Use the website's dark cinematic palette, but increase scale and contrast for video. Keep frames dense enough to feel produced: image panels, ghost type, metadata strips, registration marks, rules, and proof numbers. Do not make it feel like a web page recording.

## Rhythm

`hook-hold-fast-fast-proof-CTA`

1. Name and positioning
2. Art/culture documentary work
3. Technology/founder storytelling
4. International/cultural production range
5. Proof points and archive credibility
6. Contact and collaboration fit
